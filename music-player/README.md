# 🎵 플레이튠 (PlayTune)
> 나만의 플레이리스트를 즐기는 심플한 웹 뮤직 플레이어입니다.

**Live Demo:** [https://beencoder.github.io/javascript-project/music-player](https://beencoder.github.io/javascript-project/music-player)

---

## 💡 프로젝트 소개
좋아하는 밴드의 음악을 웹에서 바로 듣고 싶어 제작한 커스텀 뮤직 플레이어입니다. 브라우저의 기본 오디오 컨트롤러 대신 직접 디자인한 UI를 입히고, 재생/정지, 셔플, 반복 재생 등의 핵심 기능을 자바스크립트로 구현했습니다.

## 🛠 리팩토링 포인트 (Before & After)
다양한 오디오 이벤트와 플레이리스트 데이터를 효율적으로 다루는 데 집중했습니다.

* **이벤트 위임(Event Delegation) 도입**
    * 수많은 곡 리스트 아이템(`li`)에 각각 이벤트를 걸지 않고, 부모 요소에서 한 번에 이벤트를 관리하도록 개선하여 메모리 사용량을 줄였습니다.
* **시간 포맷팅 로직 공통화**
    * 재생 시간과 전체 길이를 표시할 때 중복되던 분/초 계산 로직을 `formatTime`이라는 하나의 유틸리티 함수로 통합했습니다.
* **모드 관리의 선언적 구조화**
    * 반복, 한 곡 반복, 셔플 등 복잡한 전환 로직을 `switch`문 대신 **객체 매핑** 방식으로 구현하여 코드의 가독성을 높였습니다.
* **불필요한 이벤트 등록 제거**
    * `timeupdate` 인터벌 내에서 매번 실행되던 중복 이벤트를 정리하여 성능 최적화와 잠재적인 메모리 누수를 방지했습니다.

## 🚀 주요 기능
* **커스텀 오디오 컨트롤:** HTML5 Audio API를 제어하여 재생, 일시정지, 이전/다음 곡 이동 기능을 제공합니다.
* **실시간 프로그레스 바:** 현재 재생 상태를 직관적으로 보여주며, 클릭 시 원하는 구간으로 즉시 이동(Seeking)이 가능합니다.
* **스마트 플레이리스트:** 곡 목록을 동적으로 생성하고, 현재 재생 중인 곡을 시각적으로 표시합니다.
* **다양한 재생 모드:** 한 곡 반복, 전체 반복, 무작위 셔플 기능을 통해 사용자 취향에 맞는 감상이 가능합니다.

## 🏃 실행 방법
1. 이 저장소를 클론하거나 소스코드를 다운로드합니다.
2. `assets/media/songs` 폴더에 원하는 음원 파일을 넣습니다.
3. `index.html` 파일을 브라우저에서 열고 음악을 즐겨보세요!
