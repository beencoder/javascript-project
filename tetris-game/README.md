# 🧱 미니 블록 (Mini Blocks)
> 순수 자바스크립트로 구현한 클래식 테트리스 게임입니다.

**Live Demo:** [https://beencoder.github.io/javascript-project/tetris-game](https://beencoder.github.io/javascript-project/tetris-game)

---

## 💡 프로젝트 소개
Canvas API 같은 외부 라이브러리 도움 없이, 오직 **순수 DOM 조작과 자바스크립트 로직**만으로 테트리스를 구현해 본 프로젝트입니다. 2차원 배열의 좌표 개념을 웹 화면에 매핑하는 과정에서 프로그래밍적 사고를 많이 키울 수 있었습니다.

## 🛠 리팩토링 포인트 (Before & After)
복잡한 게임 로직을 더 읽기 쉽고 효율적으로 개선하는 데 집중했습니다.

* **이벤트 처리 방식 개선 (`Key Events`)**
    * 이제는 권장되지 않는 `keyCode` 대신 `e.key`를 사용하고, `switch`문 대신 **객체 매핑(Actions Object)** 방식을 도입해 키 입력 로직을 깔끔하게 정리했습니다.
* **배열 메서드 활용 (`Every & Some`)**
    * `forEach`와 플래그 변수(`matched = true`)를 써서 복잡하게 체크하던 라인 삭제 로직을 `every` 메서드를 활용해 선언적으로 바꿨습니다. "모든 칸이 찼는가?"라는 질문을 코드에 그대로 녹였습니다.
* **나머지 연산자를 이용한 회전 로직**
    * 복잡한 조건문 대신 나머지 연산자(`%`)를 활용해 블록의 4방향 회전을 한 줄의 코드로 단순화했습니다.
* **상태 관리 구조화**
    * 게임의 설정값(CONFIG)과 실시간 상태(gameState)를 분리하여, 코드의 규모가 커져도 데이터 흐름을 쉽게 파악할 수 있도록 했습니다.

## 🚀 주요 기능
* **그리드 시스템:** 20x10 사이즈의 게임 보드를 동적으로 생성합니다.
* **충돌 감지:** 벽에 부딪히거나 다른 블록에 닿았을 때 이동을 제한하고 블록을 고정합니다.
* **라인 클리어:** 한 줄이 가득 차면 자동으로 삭제되며 점수가 올라갑니다.
* **하드 드롭:** 스페이스바를 눌러 블록을 바닥으로 즉시 내릴 수 있습니다.

## 🏃 실행 방법
1. 이 저장소를 클론하거나 소스코드를 내려받습니다.
2. `index.html` 파일을 브라우저에서 실행합니다.
3. 방향키로 블록을 조절하며 최고 점수에 도전해 보세요!
