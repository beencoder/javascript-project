---

## 💡 프로젝트 소개
과거에 자바스크립트로 간단하게 만들었던 타이핑 게임입니다. 
단순히 기능 구현에만 집중했던 예전 코드를 **가독성**과 **유지보수**를 고려해 리팩토링하며 다시 정리해 보았습니다.

## 🛠 리팩토링 포인트 (Before & After)
이번 리팩토링에서 가장 신경 쓴 부분들입니다.

* **비동기 처리 개선 (`Async/Await`)**
    * 기존의 `.then()` 체이닝 대신 `async/await`를 사용하여 코드를 위에서 아래로 읽히게 개선했어요.
* **상태 관리 모으기 (`State Object`)**
    * 여기저기 흩어져 있던 점수, 시간, 게임 상태 변수들을 `gameState`라는 객체 하나로 묶어 관리하기 시작했습니다.
* **복잡한 조건문 정리 (`Early Return`)**
    * `if`문 안에 코드를 길게 짜지 않고, 조건이 안 맞으면 바로 함수를 종료(return)시켜 로직을 단순하게 만들었습니다.
* **데이터 필터링 방식 변경**
    * `forEach`로 일일이 담던 단어 목록을 `filter` 메서드를 사용해 선언적인 코드로 바꿨습니다.

## 🚀 주요 기능
* **단어 API 연동:** 외부 API를 통해 매판 새로운 단어를 받아옵니다.
* **실시간 일치 체크:** 입력한 단어가 화면과 일치하는지 즉시 확인합니다.
* **카운트다운:** 정해진 시간 안에 단어를 입력해야 점수를 얻습니다.

## 🏃 실행 방법
1. 이 저장소를 클론(Clone)하거나 소스코드를 다운로드합니다.
2. `index.html` 파일을 브라우저에서 엽니다.
3. '게임시작' 버튼을 누르고 실력을 뽐내보세요!
